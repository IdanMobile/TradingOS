# Ambiguities -- SRC-PINE1 (Pine Bollinger Bands strategy)

1. Exit convention (upper-band touch vs. mid-band reversion) varies across published variants of this pattern; upper-band exit chosen.
2. Exact published script/author/permalink not independently fetched in this pass; grounded in the well-documented public pattern (TradingView's own Pine Script User Manual + widely-republished open-source scripts under the standard MPL-2.0 header), not one script's literal text.
3. MPL-2.0's file-level copyleft nuance does not map cleanly onto this project's 4-value license_class enum (see license_record.yaml note).
